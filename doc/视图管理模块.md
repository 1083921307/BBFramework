# 虣虣游戏框架教程(让游戏开发变得更简单)

- 同步 显示一个 View

```typescript
bb.showView(name: string,  prefab: cc.Prefab, data: any): bb.ViewCtrl;
```
---

- 异步 显示一个 View

```typescript
bb.showViewSync(pathStr: string, data: any, callback?: Function);
```
---

-  同步 显示一个 Sub View

```typescript
bb.showSubView(name: string,  prefab: cc.Prefab, data: any):  bb.ViewCtrl;
```
---

- 异步 显示一个 Sub View

```typescript
bb.showSubViewSync(pathStr: string, data: any,  callback: Function);
```
---

- 显示一个 Item

```typescript
bb.showItem(name: string,  prefab: cc.Prefab, data: any, parent: cc.Node, parentViewCtrl: bb.ViewCtrl);
```
---

-  显示一个 Item

```typescript
bb.showItemSync(pathStr: string, data: any,  parent: cc.Node, parentViewCtrl: bb.ViewCtrl, callback: Function);
```
---

- 移除一个 view

```typescript
bb.removeView(node: bb.ViewCtrl);
```
---
